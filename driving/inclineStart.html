Incline Start

Number   Steps for incline start
1) Stop where the instructor indicates without allowing the vehicle to move backwards.
2) Apply the handbrake.
3) Select the neutral (manual).
4) Check the mirrors and the blind spot.
5) Signal intention, if applicable.
6) Select gear.
7) Obtain clutch control (manual).
8) Observe 360 degrees (Rear-view mirrors and blind spot if applicable).
9) Release the handbrake.
10) Move off without rolling back.
11) Cancel the signal.
